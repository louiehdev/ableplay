AblePlay API Documentation

This page describes the backend API endpoints for games, features, and their relationships.

  • GET /api/games/{gameID}


    Returns details for a single game.
    
Example response:
{
   "id":"1234",
   "title":"Super Mario Sunshine",
   "developer":"Nintendo EAD",
   "publisher":"Nintendo",
   "release_year":2002,
   "platforms":[
      "Nintendo GameCube",
      "Nintendo Switch"
   ],
   "description":"A 2002 platforming game where Mario is framed for a crime on the tropical vacation island of 
    Isle Delfino. He must use his new companion, FLUDD (a water-squirting backpack), to clean up a mysterious 
    goop that is covering the island while also pursuing the real culprit."
}

  • GET /api/games


    Returns all games.
    
Example response:
{
    [
        {
            "id":"1234",
            "title":"Super Mario Sunshine",
            "developer":"Nintendo EAD",
            "publisher":"Nintendo",
            "release_year":2002,
            "platforms":[
                "Nintendo GameCube",
                "Nintendo Switch"
            ],
            "description":"A 2002 platforming game where Mario is framed for a crime on the tropical vacation island of 
                Isle Delfino. He must use his new companion, FLUDD (a water-squirting backpack), to clean up a mysterious 
                goop that is covering the island while also pursuing the real culprit."
        }
        {
            "..."
        }
    ]
}

    Query Parameters
    Field Type Description
    limit Integer How many results to receive

  • GET /api/games/search


    Returns all games that wildcard match the provided query.
    Query Parameters
    Field Type Description
    q String Your search query

  • POST /api/games


    Creates a new game.

  • PUT /api/games/{gameID}


    Updates an existing game.

  • DELETE /api/games/{gameID}


    Removes a game.

  • GET /api/features/{featureID}


    Returns details for one feature.
    
Example response:
{
    "id": "4321",
    "name": "Subtitles",
    "description": "Display text for dialogue and important sound effects",
    "category": "Visual"
}

  • GET /api/features


    Returns all features.
    
Example response:
{
    [
        {
            "id": "4321",
            "name": "Subtitles",
            "description": "Display text for dialogue and important sound effects",
            "category": "Visual"
        },
        {
            "..."
        }
    ]
}

    Query Parameters
    Field Type Description
    limit Integer How many results to receive

  • GET /api/features/search


    Returns all features that wildcard match the provided query.
    Query Parameters
    Field Type Description
    q String Your search query

  • POST /api/features


    Creates a new feature.

  • PUT /api/features/{featureID}


    Updates a feature.

  • DELETE /api/features/{featureID}


    Removes a feature.

  • GET /api/games/{gameID}/features/{featureID}


    Returns details for one game-feature relationship.
    
Example response:
{
    "notes": "Game provides multiple colorblind modes for visual assistance",
    "verified": true,
    "title": "Baldur's Gate 3",
    "name": "Colorblind Modes"
}

  • GET /api/games/features


    Returns all games with their features.
    
Example response:
{
   [
      {
         "id":"1234",
         "title":"Super Mario Sunshine",
         "developer":"Nintendo EAD",
         "publisher":"Nintendo",
         "release_year":2002,
         "platforms":[
            "Nintendo GameCube",
            "Nintendo Switch"
         ],
         "description":"A 2002 platforming game where Mario is framed for a crime on the tropical vacation island of 
          Isle Delfino. He must use his new companion, FLUDD (a water-squirting backpack), to clean up a mysterious 
          goop that is covering the island while also pursuing the real culprit.",
         "game_features":[
            {
                "id": "4321",
                "name": "Colorblind Modes",
                "notes": "Game provides multiple colorblind modes for visual assistance",
                "verified": true
            },
            {
                "..."
            }
         ]
      },
      {
         "..."
      }
   ]
}

    Query Parameters
    Field Type Description
    limit Integer How many results to receive

  • GET /api/games/{gameID}/features


    Returns features for one game.
    
Example response:
{
   [
      {
         "notes":"Subtitles exist and work in this game",
         "verified":true,
         "feature_id":"4321",
         "name":"Subtitles",
         "description":"Display text for dialogue and important sound effects",
         "category":"Visual"
      },
      {
         "..."
      }
   ]
}

  • GET /api/features/{featureID}/games


    Returns games with a specific feature.
    
Example response:
{
   [
      {
         "notes":"Subtitles exist and work in this game",
         "verified":true,
         "feature_id":"4321",
         "game_id":"1234",
         "title":"Super Mario Sunshine",
         "developer":"Nintendo EAD",
         "publisher":"Nintendo",
         "release_year":2002,
         "platforms":[
            "Nintendo GameCube",
            "Nintendo Switch"
         ],
         "description":"A 2002 platforming game where Mario is framed for a crime on the tropical vacation island of 
          Isle Delfino. He must use his new companion, FLUDD (a water-squirting backpack), to clean up a mysterious 
          goop that is covering the island while also pursuing the real culprit.",
      },
      {
         "..."
      }
   ]
}

  • POST /api/games/{gameID}/features


    Associates a feature to a game.

  • PUT /api/games/{gameID}/features/{featureID}


    Updates relationship notes / verification.

  • DELETE /api/games/{gameID}/features/{featureID}


    Removes a feature from a game.
