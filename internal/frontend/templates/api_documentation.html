AblePlay API Documentation

This page describes the backend API endpoints for games, features, and their relationships.

  • GET /api/games
    Returns all games.
  • GET /api/games/{gameID}
    Returns details for a single game.
  • POST /api/games
    Creates a new game.
    {"title": "Example", ...}
  • PUT /api/games/{gameID}
    Updates an existing game.
  • DELETE /api/games/{gameID}
    Removes a game.

  • GET /api/features
    Returns all features.
  • GET /api/features/{featureID}
    Returns details for one feature.
  • POST /api/features
    Creates a new feature.
  • PUT /api/features/{featureID}
    Updates a feature.
  • DELETE /api/features/{featureID}
    Removes a feature.

  • GET /api/games/features
    Returns all games with their features.
  • GET /api/games/{gameID}/features
    Returns features for one game.
  • GET /api/features/{featureID}/games
    Returns games with a specific feature.
  • POST /api/games/{gameID}/features
    Associates a feature to a game.
  • PUT /api/games/{gameID}/features/{featureID}
    Updates relationship notes / verification.
  • DELETE /api/games/{gameID}/features/{featureID}
    Removes a feature from a game.
