Game Feature Relationships

GET /api/games/features

Returns all games with their associated features.

GET /api/games/{gameID}/features

Returns the features associated with a specific game.

GET /api/features/{featureID}/games

Returns all games associated with a specific feature.

GET /api/games/{gameID}/features/{featureID}

Returns the detailed relationship record between one game + one feature (e.g., notes).

POST /api/games/{gameID}/features

Associates a feature with a game.

Example JSON Body:

{
        "feature_id": "",
        "notes": ""
        }
